# Closure Table

## 新增 (我是新人)

### Node 表：新增我。

### Path 表：

- 找到「誰是爸爸的長輩」，複製一份，後代填「我」。
- 加一筆「我到我」的紀錄。

## 刪除 (我跟我這房都消失)

### Path 表：

- 找出所有「我是長輩」的紀錄（這會抓到我和我所有的子孫）。
- 把這些人的所有關係紀錄全部刪掉。

### Node 表：

- 把我和我的子孫通通刪掉。

## 移動 (我帶著子孫改嫁)

### Path 表 - 第一步 (撕合約)：

- 把所有「我的舊長輩」跟「我及我的子孫」之間的合約全部撕掉。

### Path 表 - 第二步 (簽新約)：

- 讓「新爸爸的所有長輩」跟「我及我的子孫」簽新合約。
- 註：我跟我子孫之間的舊合約不必動，因為我們內部的長幼順序沒變。
